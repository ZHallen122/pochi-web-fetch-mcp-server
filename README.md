# web-fetch-mcp

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run build
```

```bash
bun run start
```

# pochi-web-fetch-mcp-server

### dev

```bash
npx wrangler dev
```

### deploy

```bash
wrangler secret put JINA_TOKEN
```

```bash
wrangler secret put PORT
```

```bash
wrangler deploy
```
